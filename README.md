# TECHIN510 Lab 1
Veronika Sermeno Pon

This Streamlit app is a system where GIX students can submit purchase requests for their course projects. It takes the same
user inputs of GIX's Excel Sheet into an online form. Students use the Student portal to add item information, instructor and coordinator
uses Staff portal to approve and notify student if approved and estimated delivery dates. 

# Setup Instructions

This README.md instructions was was written with the assistance of ChatGPT 

Follow these steps to get the project running locally:

## 1. Clone the Repository

```bash
git clone https://github.com/vpon-lab/TECHIN510_Lab1.git
cd TECHIN510_Lab1
```

## 2. Create a Virtual Environment

```
python3 -m venv .venv
source .venv/bin/activate
```

## 3. Install Dependencies 
```
pip install -r requirements.txt
```
